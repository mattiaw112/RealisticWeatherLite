# FS25_Realistic Weather Lite

## Description / Descrizione
**EN:** This mod adds a lightweight and optimized weather management system, inspired by the original "Realistic Weather" mod by Arrow. It focuses on essential effects like dynamic fog, snow accumulation, and hail damage, excluding the complex separate humidity system.

**IT:** Questa mod aggiunge un sistema di gestione meteo leggero e ottimizzato, ispirato alla mod originale "Realistic Weather" di Arrow. Si concentra sugli effetti essenziali come nebbia dinamica, accumulo di neve e danni da grandine, escludendo il complesso sistema di umidità separato.

## Features / Caratteristiche
* **Dynamic Fog:** Realistic visibility changes.
* **Snow Accumulation:** Improved snow buildup logic.
* **Hail Damage:** Vehicles exposed to hail will suffer damage.
* **Lightweight:** Does not replace core game files, but hooks into the original system.
